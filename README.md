## What It Does

ApplyPilot scrapes jobs from across the web, scores them against your profile with two passes (fast keyword scoring + Claude AI re-scoring), and puts the best matches in front of you. You review and apply on your own terms.

```bash
pip install applypilot
pip install --no-deps python-jobspy && pip install pydantic tls-client requests markdownify regex
applypilot init          # one-time setup: resume, profile, preferences, API keys
applypilot doctor        # verify your setup
applypilot run discover enrich fastscore llmscore   # scrape + score
applypilot review        # open top-scoring jobs in your browser
applypilot review --score 8   # open all 8s, then 7s, etc.
applypilot portland      # Portland/Oregon area jobs by score
applypilot browse        # open Greenhouse/Lever jobs for manual apply
```

> **Why two install commands?** `python-jobspy` pins an exact numpy version in its metadata that conflicts with pip's resolver, but works fine at runtime with any modern numpy. The `--no-deps` flag bypasses the resolver; the second command installs jobspy's actual runtime dependencies.

## The Pipeline

| Stage | What Happens |
|-------|-------------|
| **1. Discover** | Scrapes Indeed + LinkedIn via JobSpy, 12+ Workday employer portals, and 36 Greenhouse company boards |
| **2. Enrich** | Fetches full job descriptions via JSON-LD, CSS selectors, or AI-powered extraction |
| **3. fastscore** | Keyword-based scoring (fast, no LLM) — matches your tech stack against the job description |
| **4. llmscore** | Claude Haiku re-scores every job: title nuance, YOE buried in prose, stack fit, role discipline |
| **review** | Opens jobs by score tier in your browser — start with 10s, then 9s, etc. |

Each stage is independent. Run them all or pick what you need.

## Requirements

| Component | Required For | Details |
|-----------|-------------|---------|
| Python 3.11+ | Everything | Core runtime |
| Gemini API key | Scoring | Free tier (15 RPM / 1M tokens/day) is enough |
| Claude Code CLI | llmscore | Install from [claude.ai/code](https://claude.ai/code) |

**Gemini API key is free.** Get one at [aistudio.google.com](https://aistudio.google.com).

## Configuration

All generated by `applypilot init`:

### `profile.json`
Your personal data: contact info, work authorization, compensation, experience, skills. Powers scoring and filtering.

### `searches.yaml`
Job search queries, target titles, locations. Configure which Workday employers and Greenhouse boards to scrape.

### `.env`
API keys: `GEMINI_API_KEY`, `LLM_MODEL`.

### Package configs (shipped with ApplyPilot)
- `config/employers.yaml` - Workday employer registry
- `config/sites.yaml` - Greenhouse boards and manual-apply sites
- `config/searches.example.yaml` - Example search configuration

## CLI Reference

```
applypilot init                              # First-time setup wizard
applypilot doctor                            # Verify setup, diagnose missing requirements
applypilot run discover enrich fastscore llmscore   # Full scrape + score run
applypilot run --workers 4                   # Parallel discovery/enrichment
applypilot run --rescore                     # Re-score all jobs (fastscore only)
applypilot run --dry-run                     # Preview without executing
applypilot review                            # Open top score tier in browser
applypilot review --score 9                  # Open all jobs scored exactly 9
applypilot review --min-score 7              # Open all jobs scored 7+
applypilot portland                          # Portland/Oregon jobs by score
applypilot browse                            # Open Greenhouse/Lever jobs (manual-apply)
applypilot status                            # Pipeline statistics
applypilot dashboard                         # Open HTML results dashboard
```
